Link to github url to see the full code. Go to Git

Steps to regenerate the application

  1. Download or clone from above URL
  2. Navigate to folder in CMD and type 'npm install'
  3. Type 'ng serve'
  4. Let the application bundle and complie
  5. Go to browser and type 'localhost:4200'

Note: validations are not implemented so clicking on add button will always work and create a new row.

By default scrollbar is hidden for Campaign Management but still scrolls :)
